# DEDSECIP is a simple IP Tracker that built by Python. Using a free API to get some detail informations about IP Address.

# USAGE
```
sudo python dedsecip.py -t 127.0.0.1
```
# INSTALL
```
git clone https://github.com/0xbitx/dedsecip-.git 
cd dedsecip-
```
